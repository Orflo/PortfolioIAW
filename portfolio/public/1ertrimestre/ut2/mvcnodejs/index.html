Arquitectura MVC en Node.JS

Descripción de la práctica

Objetivo de la práctica

El objetivo fue crear una Arquitectura MVC con Node.JS.

Procedimientos seguidos

Las intrucciones que nos indicó el profesor y el manual asignado a la práctica.

Resultados obtenidos

Se creó una carpeta todo-app y en su interior se generaron directorios al ejecutar el comando npm init -y y el comando npm install express ejs de Node.JS y se creó a continuación la siguiente estructura:

/todo-app
    /models
        taskModel.js
    /views
        index.ejs
        add.ejs
    /controllers
        taskController.js
    server.js
Contenido de los archivos .js y .ejs
let tasks = []; // Simulando base de datos con un array
module.exports = {
getAll: () => tasks,
add: (task) => tasks.push(task),
delete: (index) => tasks.splice(index, 1)
};
<!-- index.ejs -->
<h1>Tasks</h1>
<ul>
    <% tasks.forEach((task, index) => { %>
        <li>
            <%= task %>
            <a href="/delete/<%= index %>">Delete</a>
        </li>
    <% }); %>
</ul>
<a href="/add">Add Task</a>
<!-- add.ejs -->
<h1>Add Task</h1>
<form action="/add" method="post">
    <input type="text" name="task" required>
    <button type="submit">Add</button>
</form>
const express = require('express');
const router = express.Router();
const taskModel = require('../models/taskModel');

router.get('/', (req, res) => {
    const tasks = taskModel.getAll();
    res.render('index', { tasks });
});

router.get('/add', (req, res) => {
    res.render('add');
});

router.post('/add', (req, res) => {
    taskModel.add(req.body.task);
    res.redirect('/');
});

router.get('/delete/:id', (req, res) => {
    taskModel.delete(req.params.id);
    res.redirect('/');
});
    
module.exports = router;
const express = require('express');
const taskRoutes = require('./controllers/taskController');
const app = express();
const PORT = 3000;

app.set('view engine', 'ejs');
app.use(express.urlencoded({ extended: true })); // Middleware para procesar datos de formularios

app.use('/', taskRoutes);

app.listen(PORT, () => {
    console.log(`Server running at http://localhost:${PORT}`);
});

El resultado obtenido fue satisfactorio, ya que nos permitía simular una app simple para añadir tareas y luego eliminarlas.

Conclusiones

Está bien para entender como es la estuctura de una app simple y como funciona la Arquitectura MVC.
